Troubleshooting

If you can't hear the audio:

  • Make sure your device's volume is turned up
  • Check that the audio files are in the correct location (sounds folder)
  • Try using different audio formats (MP3, WAV, OGG) if one doesn't work
  • Some mobile browsers require user interaction before playing audio
